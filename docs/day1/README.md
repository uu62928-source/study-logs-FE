# 1日目 ドキュメント案内

1日目のテーマは「アーキテクチャ設計と境界づけ」です。

Reactアプリを画面の集まりとしてではなく、変更理由と依存方向に基づいて分割する方法を学びます。Feature-based ArchitectureとClean Architectureを、現在の学習ログアプリへ適用しながら整理しています。

## ドキュメント

### 1. [学習ガイド](./study-guide.md)

1日目の目的、必要な知識、完了条件、学習の進め方をまとめています。

学習を始める前に読み、今日何を理解できればよいか確認するためのドキュメントです。

### 2. [学習ノート](./learning-notes.md)

学習中に分かりにくかった点と、今後も使える判断基準をまとめています。

Feature-based Architecture、依存方向、`shared`へ移動する条件、Clean Architectureなどの詳しい解説を確認できます。

### 3. [振り返り](./retrospective.md)

1日目に学んだこと、採用した設計方針、見送った方針、理解に時間がかかったことをまとめています。

学習後に読み返し、自分の言葉で説明できるか確認するためのドキュメントです。
